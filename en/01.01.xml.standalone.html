01.01 Rehearse HTML and CSS

Difficulty: easy
Size: XL


Create a website about yourself OR about some city you know. Create it inside your public_html folder on the Myy-server (M:\public_html). Call it JoesWebsite, where you replace Joe with your own name. Create there 2-3 extremely short web pages where all the style is in CSS files. Create subfolders for CSS and images. Use maximum 90 minutes for this task. A bit more if you didn’t do the previous HTML+CSS course project yet. Error-free and technically correct websites are appreciated. Do not spend too much time on graphical design and write as short HTML and CSS as possible. Make all the pages and CSS valid and add an https://validator.w3.org/ validator link logo to the pages.

HINTS:

More information from this document, if needed: http://myy.haaga-helia.fi/~valju/common/WhatToKnowBeforeJavaScriptClientSideProgramming.pdf

What to submit: A link to the main page of your website, for example http://myy.haaga-helia.fi/~a1234567/JoesWebsite/index.html
