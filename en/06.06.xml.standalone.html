06.06 Conditions

Difficulty: medium
Size: S


When are the following conditions evaluated as true?

  1. if (dicePips == 1 || dicePips == 3 || dicePips == 5)
  2. if (dicePips == 1 && dicePips == 3 && dicePips == 5)
  3. if (age > 18 && age < 65)
  4. if (age > 18 || age < 65)
  5. if (age > 18 && income > 7353)
  6. if (age > 18 || income > 7353)
  7. if (isNaN(age) == true)
  8. if (isNaN(age))
  9. if (isNaN(age) == false)
  10. if (! isNaN(age))

HINT: The isNaN(value) function returns true, if the value is Not-A-Number. Otherwise the function returns false.
