07.04 Parameters and Return Value

Difficulty: easy
Size: S


The same function can accept input parameters and return a value.

Assign an onclick event to the 'ROLL THE DICE' button so that the function rollTheDice() is called when the button is clicked.

The function rollTheDice() should do the following:

  • Call the function randomInteger(from, to) with parameter values 1 and 6.
  • Assign the return value of the function to the variable diceValue.
  • Insert the value of the variable diceValue into the HTML div element whose id is 'dice'.

Test your code. Each time you click the button you should see a random value between 1 and 6 below the button.
