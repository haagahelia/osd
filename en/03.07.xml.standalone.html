03.07 Eclipse IDE

Difficulty: medium
Size: M


IDE (Integrated Development Environment) is single software application in which all development is done. Typically, an IDE provides a single user interface for managing source codes and other files, servers, browsers and other programmer's tools that speed up the development work.

  1. Start Eclipse in either Windows (C:\install\eclipse..\eclipse.exe (Autumn 2016)) or Linux desktop.
  2. Create folder eclipse_workspace inside Myy’s oswd-folder and choose it as the workspace.
  3. Close the Welcome tab.
  4. Create a new project in Eclipse as follows:
    File > New > Project… > General > Project
    Click on Next
    Enter MyFirstEclipseProject as the project name and click on Finish.
  5. Create the following two folders in the project: css and js
    In the Project Explorer tab, right-click on the project name
    New > Folder
  6. Create three new files (general.css, main.js, myPage.html) as below.
  7. Copy/paste the following code to the file myPage.html: 
    
 <!DOCTYPE html>
 <html>
 <head>
   <title>My Page </title>
   <link rel="stylesheet" type="text/css" href="css/general.css">
 </head>
 <body>
   <h1 id="myTitle">Hello </h1>
   <script type="text/javascript" src="js/main.js"> </script>
 </body>
 </html>
  8. Copy/paste the following code to the file general.css: 
    #myTitle {
    color: #0055AA;
}
  9. Copy/paste the following code to the file main.js: 
    document.getElementById("myTitle").innerHTML += ", World!";
  10. View the HTML page in web browser as follows:
    In Project Explorer, right-click on the file name myPage.html
    Select Open with > Web Browser
Finally, take a screenshot. Save the screenshot to Myy's public_html folder and submit a link to the screenshot.
