02.03 BMI using Blocklies

Difficulty: medium
Size: XL


Please create a Body Mass Index (BMI) calculator using Blocklies. The BM-index is calculated with the formula: BMI = weight in kilograms / (height in meters * height in meters). So e.g. if you weigh 70 kilograms and you are 1.8 meters tall, then your BMI is 70 / (1.8 * 1.8) = 22

Your application first asks the user for his name, then for his weight and height. Then it shows a message telling: "Hi name! Your body mass index is x"

From this URL you can find all the blocks you'll need for creating the application: https://blockly-demo.appspot.com/static/demos/code/index.html#nmni35

Here is a video instructions:https://youtu.be/Od_wHc7_kT0
