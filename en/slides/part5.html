VARIABLES

CONCEPT

How to visualize a variable?

USING VARIABLES

// create a variable
var age;
// store a value
age = 23;
// change the value
age = 24;
// read the value and log it
console.log(age);

TYPES

  • Number
  • String
  • Array
  • Object

Example

var length = 16;                                  // Number
var lastName = "Johnson";                         // String
var carNames = ["Saab", "Volvo", "BMW"];          // Array
var person = {firstName:"John", lastName:"Doe"};  // Object
