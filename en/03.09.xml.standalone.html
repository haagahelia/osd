03.09 (Extra: Log in to Linux and Mount Myy)

Difficulty: easy
Size: S


First, open the page https://ondemand.cp.haaga-helia.fi and log in to the CentOS Linux environment.


You can access the ondemand-system directly from Haaga-Helia campus network. Use Internet Explorer, because Citrix Reciever plugin is preinstalled in IE. If you are connecting from an external network, open first VDI desktop from http://vdi.haaga-helia.fi and from there connect the ondemand system.


Double-click the icon "myy.sh" on your CentOS desktop and select "Run" (do not choose "Run in Terminal"). Type your password into the opening window and check that the directory /mnt/myy contains now your files from the Myy-server.


If you can't see any files in the folder /mnt/myy, please open Terminal app and run the following commands: "fusermount -u /mnt/myy" to unmount the broken folder and run "sshfs yourstudentnumber@myy.haaga-helia.fi:/u10/yourstudentnumber /mnt/myy" to mount it again. Naturally, replace yourstudentnumber with your actual student number. If you are not a student, but a faculty member, please use remote folder "/homeh/yourusername" instead.


Then take a screenshot of the Linux desktop and save the screenshot image to your public_html folder on the Myy server.


Finally, submit a link to the screenshot image.


CentOS will be reseted every night, so don’t save anything important to the local hard drive of CentOS.
