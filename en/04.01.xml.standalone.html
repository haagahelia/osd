04.01 Scripts in <head> and <body>

Difficulty: easy
Size: S


JavaScript can be placed in the <head> and the <body> sections of an HTML page. The script location is significant.

Place the following script in the <head> section of the HTML page: 

<script>document.write("A");</script>
Place the following header element in the <body> section: 
<h1>Hello</h1>
Place the following script in the <body> section, after the header element: 
<script>document.write("B");</script>
What is shown on the page and in which order? How does the location of the JavaScript code affect to the result?
