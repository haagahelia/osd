03.01 Download Brackets

Difficulty: easy
Size: S

In classroom, open the Brackets editor (C:\Program Files(x86)\Brackets\brackets.exe (Autumn 2016)). Take a screenshot of the Brackets editor and save the screenshot image to your public_html folder on the Myy server.

At home, download Brackets Portable code editor (.exe) from the page https://github.com/sagiegurari/brackets-portable/releases/tag/1.6.0 to c:\install. See more information about Brackets: http://brackets.io/.

Finally, submit a link to the screenshot image.
