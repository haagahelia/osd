03.06 JSBin

JSBin is a handy tool for trying out some JavaScript code and sharing the code with others.

  1. Open the page http://jsbin.com and close the light blue ad on the top of the page.
  2. If you do not have a JSBin username yet, then click on 'Login or Register' to create a username for yourself. By logging in, you can later access all your previous code that is saved on the JSBin site.
  3. Next, you create a new Bin. Make sure that 'Auto-run JS' is checked on the Output tab.
  4. In the HTML tab, write a new <h1> header element into <body> as follows: 
    <h1 id="myTitle">Hello</h1>
  5. Open the CSS tab. In the CSS tab, write the following code to change the color of the text in the myTitle element: 
    #myTitle {
    color: #0055AA;
}
  6. Open the JavaScript tab. In the JavaScript tab, write the following code to programmatically append more text to the myTitle element: 
    document.getElementById("myTitle").innerHTML += " world!";

Finally, submit a link to your Bin. Below is an example of a link to a Bin.
http://jsbin.com/asdfqwerty/edit?html,css,js,output
