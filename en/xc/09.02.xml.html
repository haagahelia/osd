09.02 Sort Array

First, sort the given array in reverse alphabetical order (z-a). After that, write all the array element values into the browser console. Each value should be displayed on a separate line.

The Array:

var array = ["Milk", "Bread", "Juice", "Eggs", "Butter", "Salt", "Oranges", "Apples"];

HINTS:

  • The sort() method sorts the elements of an array in ascending order. Example: 
    words.sort();
  • The reverse() method reverses the order of the elements in an array. Example: 
    words.reverse().
