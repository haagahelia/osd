04.04 HTML Manipulation

In this exercise, you change the content of an HTML element when the user clicks a button

  1. Add a new button tag to your HTML document. The button tag should be inside <body%gt;.
    2. The text shown on the button should be "CLICK ME". When the user clicks the button the function pressed() should be called.
  2. Add a <p> element below the button. The id of the element should be "feedback".
  3. In your JavaScript code, write a function called pressed(). When the function is called it should set the text "THANKS:)" to the <p> element.
HINT: The following statement writes "Hello, dude!" inside an HTML element with id "myElement": 
document.getElementById("myElement").innerHTML = "Hello, dude!";
