03.02 Use Brackets Code Editor

Difficulty: easy
Size: S

In the Windows environment, use the Brackets code editor to create a new HTML page with the name secondpage.html. Write required html-tags and a paragraph with text 'Written with Brackets'. Save the HTML page to the oswd folder on the Myy server. When the HTML page is opened in a web browser, it should show the text 'Written with Brackets'. Take a screenshot of the code editor window and save the screenshot image to the public_html folder on the Myy server.

Finally, submit a link to the screenshot image (for example http://myy.haaga-helia.fi/~yourstudentnumber/brackets_screenshot.png).
