03.02 Use Text Editor

In the Linux environment, use the gedit text editor to create a new HTML page with the name second.html. Save the HTML page to the oswd folder on the M:-drive. When the HTML page is opened in a web browser, it should show the text 'Written with gedit'. Take a screenshot of the text editor window and save the screenshot image to the public_html folder on the M:-drive.

Finally, submit a link to the screenshot image.
