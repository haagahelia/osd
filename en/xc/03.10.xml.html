03.10 (Extra: Use Text Editor)

Difficulty: easy
Size: S

In the Linux environment, use the gedit text editor to create a new HTML page with the name second.html. Write required html-tags and a paragraph with text 'Written with gedit'. Save the HTML page to the oswd folder on the Myy server. When the HTML page is opened in a web browser, it should show the text 'Written with gedit'. Take a screenshot of the text editor window and save the screenshot image to the public_html folder on the Myy server.

Finally, submit a link to the screenshot image (for example http://myy.haaga-helia.fi/~yourstudentnumber/gedit_screenshot.png).
