04.02 External Script File

In most cases, it is recommendable to place JavaScript code in external files. This has several advantages. When JavaScript and HTML codes are separated they are easier to read, maintain and reuse. In addition, this might speed up page loads in some cases

Place the following HTML code in the <body> section of the HTML document: 

 <pre><code class="language-css">#myDiv {
  border-width: 3px;
  border-style: solid;
}</code></pre>

Place the following CSS file reference in the <head> section of the HTML document: 


<link rel="stylesheet" type="text/css" href="https://cdnjs.cloudflare.com/ajax/libs/prism/0.0.1/prism.min.css" />

Place the following external JavaScript file reference in the <head> section of the HTML document: 


<script src="https://cdnjs.cloudflare.com/ajax/libs/prism/0.0.1/prism.min.js"></script>
