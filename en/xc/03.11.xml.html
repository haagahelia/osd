  • Step 1. In the Linux environment, open the Terminal Window.
  • Step 2. In the Linux Terminal Window, go to Myy's oswd folder (cd /mnt/myy/oswd). Then write the command python -m SimpleHTTPServer and press Enter to start a simple web server.
  • Step 3. In the Linux environment, open the page http://localhost:8000/second.html in a web browser.
  • Step 4. In the Linux Terminal Window, you see some text that the web server has displayed. What does the text say? Finally, submit a copy of the text.
  • In the Linux Terminal Window, you can close the web server by pressing Ctrl+c.
