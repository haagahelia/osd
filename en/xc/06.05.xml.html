06.05 Body Mass Index

Difficulty: easy
Size: M

Make a program that asks for the user’s weight and height and calculates, and tells the body mass index class.

Sample output for shown input:

The body mass index is BMI = weight / (height/100.0 * height/100.0).
Weight less than normal weight 18,4 or less
Normal weight18,5 - 24,9
Overweight25,0 or more

HINTS:

Selection structure follows this idea: 


if (bmi <= 18.4) {
    // Weight less than normal weight 
} otherwise if (bmi <= 24.9) { 
    // Normal weight
}otherwise { 
    // Overweight 
}
