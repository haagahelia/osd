08.04 Same Pips

Make a program that randomizes two dice pips (noppaluku) until both pips are the same. At the end the program tells how many randomization rounds were needed.

  • Write the code to a script element that you place inside the body element.
  • Initialize the count variable to be zero. That variable is used to count the randomization rounds.
  • Make a loop inside of which you randomize the two pips values until they will be the same. Use the while loop.
