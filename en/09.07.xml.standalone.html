09.07 Images

Difficulty: hard
Size: M


Make a program, that shows all the images in the array.

Sample output:

The (text) array is 

var imageNames = ["face.png", "heart.png", "sun.png", "moon.png", "cloud.png"];
The ready-made images files are at https://myy.haaga-helia.fi/~swd1ta001/kuvat/. Save the images in the same folder as the html page that contains the JavaScript. You may also add you own image files as long as you also add the file name to the name array.

HINTS:

Use a for loop to show all the images.
Use the document.write() method to write one new image element to the web page per each image. 

<body>
   <script type="text/javascript">
      for (var i = 0; i < imageNames.length; i++) { 
           // Write here one more image element on the page 
      }
   </script>
</body>
The image element should be like 
<img src='name-of-the-image-file' alt='Funny shape'>.
E.g. 
<img src='face.png' alt='Funny shape'>
